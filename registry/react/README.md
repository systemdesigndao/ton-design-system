0010_2, 2_10 - Available to copy via CLI

3. Lazy Routes (0100 | 4, 0101 | 5)

## Installation

Install the necessary dependencies via npm:

```bash
ppnpm install
```

To run the project in development mode, use the command:

```bash
pnpm run dev
```

For building the project, use:

```bash
pnpm run build
```
