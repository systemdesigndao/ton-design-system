## Use CLI

### Quick start

1. Copy component from Github - `npx "@designervoid/ton-design-system"@"latest" cli -g`
2. Init Tailwind `npx "@designervoid/ton-design-system"@"latest" cli -t`
3. Init TDS `npx "@designervoid/ton-design-system"@"latest" cli -tds`
